综述
胱抑素 C 在糖尿病及其血管并发症中的研究进展
王咏波  王景红  杜建玲
[提要]   胱抑素 C 是一种半胱氨酸蛋白酶抑制剂,广泛存在于各种组织的有核细胞和体液中,主要功
能是抑制内源性半胱氨酸蛋白酶的活性。 研究发现,胱抑素 C 能够预测糖尿病的发生,并与糖尿病大血管
及微血管并发症如蛋白尿、视网膜病变、外周动脉疾病、冠状动脉疾病密切相关。 本文将对胱抑素 C 在糖
尿病及其血管并发症研究中的进展进行综述。
[关键词]   胱抑素 C; 糖尿病; 动脉粥样硬化; 糖尿病肾病; 糖尿病视网膜病变
Research progress of cystatin C in diabetes mellitus and its vascular complications Wang Yongbo, Wang
Jinghong, Du Jianling. Department of Endocrinology, the First Affiliated Hospital of Dalian Medical University,
Dalian 116011, China
[Summary]  Cystatin C is widely present in nucleated cells of various tissues and body fluids as a cysteine
protease inhibitor. It can predict the occurrence of diabetes, and it was significantly associated with diabetic
macrovascular and microvascular complications such as albuminuria, retinopathy, peripheral arterial disease, and
coronary artery disease. This review introduces the research progress of cystatin C in diabetes and diabetic vascular
complications.
[Key words]  Cystatin C; Diabetes mellitus; Atherosclerosis; Diabetic nephropathy; Diabetic retinopathy
(Chin J Endocrinol Metab, 2014, 30: 859-861)
胱抑素 C 即半胱氨酸蛋白酶抑制蛋白 C,是一种低分子量
蛋白质,研究显示,胱抑素 C 不仅是早期肾损害的标志,还能够
预测糖尿病的发生,参与糖尿病合并冠心病、外周动脉疾病和
糖尿病肾病等糖尿病大血管及微血管病变的发生和发展。 本
文主要就胱抑素 C 在糖尿病及其血管并发症中的研究进展作
一综述。
一、 胱抑素 C 的分子生物学特性和功能
胱抑素 C 又被称作 γ-痕迹蛋白、γ-后球蛋白,是一种低分
子量(13. 3 kD)、碱性非糖化蛋白质,是半胱氨酸蛋白酶抑制剂
家族中的一员。 人的胱抑素 C 基因片段位于 20 号染色体短臂
13 区 2 带,大约为 413 kB,包含 3 个外显子,2 个内含子。 编码
胱抑素 C 的基因属于管家基因,能在所有的有核细胞内以恒定
速度持续转录与表达,无组织特异性。 因此,胱抑素 C 广泛存
在于各种组织的有核细胞和体液中,精液、脑脊液中浓度较高,
血浆中次之,尿液中浓度最低。 以前认为,血中胱抑素 C 浓度
不受性别、年龄等因素的影响,在 1 岁以后其浓度稳定在成人
水平,一直到 60 岁左右。 但近年来的研究表明,胱抑素 C 受性
别、年龄、甲状腺功能、糖皮质激素等多种因素影响,男性、高
龄、吸烟、高血压、肥胖、炎症、脂代谢紊乱、低白蛋白血症、甲状
腺功能亢进、应用糖皮质激素等均与胱抑素 C 升高有关[1,2] ,其
中年龄对胱抑素 C 水平的影响最大,80 岁以上人群血清胱抑
素 C 升高(>1. 12 mg / L)者超过 50% [1] 。
在生理条件下,胱抑素 C 的主要功能是抑制内源性半胱氨
酸蛋白酶的活性,其中对组织蛋白酶 B 的抑制作用最强,参与
细胞内肽类和蛋白质代谢,尤其是胶原代谢,维护细胞外基质
的动态平衡。 病理条件下,胱抑素 C 含量减少时,组织蛋白酶
活性相对增强,造成胶原分解、弹性纤维断裂,从而导致细胞外
基质的重塑和血管壁的重构,造成动脉粥样硬化[3] ;可能还参
与炎症和肿瘤等病理过程[4] ;另外,胱抑素 C 与糖尿病[5,6] 及
其慢性并发症[7]如蛋白尿、视网膜病变、外周动脉疾病、冠状动
脉疾病密切相关。
二、 胱抑素 C 与糖尿病
研究表明,胱抑素 C 可预测糖尿病的发生。 Donahue 等[5]
对 1 455 例血糖正常且没有心血管疾病者进行 6 年的随访,结
果发现 91 例发生了空腹血糖升高,且同时伴有胱抑素 C 升高,
胱抑素 C 高表达的人群其发生血糖升高的风险是正常人的 3
倍;Logistic 回归分析发现,校正血糖、吸烟、肥胖、年龄等相关危
险因素后,胱抑素 C 水平升高者其进展为糖尿病前期的风险会
显著增加,进一步校正超敏 C 反应蛋白(hs-CRP)、白介素-6、E-
选择素等炎症因子后,这种风险仍然存在,提示胱抑素 C 是预
测空腹血糖升高最强烈的独立危险因素,说明胱抑素 C 有可能
成为预测糖代谢异常的指标。 同样,Sahakyan 等[6] 对 3 472 名
非糖尿病者随访 15 年也发现,血清胱抑素 C 水平与 2 型糖尿
病的发病率独立相关。 Surendar 等[8] 的研究发现,糖耐量正常
者中代谢综合征(MS)危险因素越多,血清胱抑素 C 水平越高,
随着 MS 危险因素的减少,胱抑素 C 水平直线下降;而合并 MS
危险因素越多,发生糖耐量受损的可能性就越大,提示胱抑素
C 可能参与血糖正常者向糖尿病前期进展的过程。
目前 胱 抑 素 C 参 与 糖 尿 病 的 发 病 机 制 尚 不 明 确。
Sahakyan 等[6]研究显示,糖尿病者较非糖尿病者具有较高水平
的胱抑素 C 和较低水平的由胱抑素 C 估算的肾小球率过滤
(eGFR胱抑素C)。 故推断胱抑素 C 可能通过特异性反映慢性肾
脏病而直接或间接影响糖尿病的发生。 但在无肾脏疾病的人
群中经校正年龄、性别、体重指数(BMI)等因素后,胱抑素 C 与
糖尿病的发生仍有显著关系,当进一步校正 HBA1C后,这种关
系被弱化,提示胱抑素 C 也可能通过其他因素的参与而影响糖
尿病的发生[6] 。 Reutens 等[9]研究显示,血清胱抑素 C 与 2 型
糖尿病发病率的相关性独立于血肌酐 ( SCr)、空腹血糖和
HBA1C,却受 BMI、腰围或胰岛素抵抗指数的影响;胱抑素 C 仅
与 BMI、腰围或胰岛素抵抗指数≥中位数的人群糖尿病发病率
相关,故推断胱抑素 C 参与糖尿病的发生可能与中心性肥胖或
胰岛素抵抗有关。
三、 胱抑素 C 与糖尿病大血管病变
Maahs 等[10]研究显示,胱抑素 C 在 T1DM 患者中能预测亚
临床冠状动脉粥样硬化(SCA)的进展,即使校正了其他危险因
素,这个结论仍然成立,将胱抑素 C 作为 SCA 预测因子,并将
胱抑素 C 与心血管事件及死亡率的关系拓展至糖尿病人群。
随后他们发现胱抑素 C 对冠脉粥样硬化进展的影响在 T1DM
和非糖尿病人群中无明显差异,但出乎意料的是在肾功正常的
非糖尿病人群中高水平的胱抑素 C 可延缓冠状动脉粥样硬化
的进程,提示高水平胱抑素 C 对动脉有保护作用[11] 。 西班牙
一项研究发现,2 型糖尿病合并冠心病的患者血清胱抑素 C 水
平高于 2 型糖尿病无冠心病组及对照组[12] 。
金启辉等[12]研究显示,初发 2 型糖尿病在肾功正常,SCr、
血脂(除甘油三酯外)无差异的条件下,颈动脉粥样硬化组胱抑
素 C 水平显著高于非颈动脉粥样硬化组;Spearman 相关分析发
现胱抑素 C 水平与双侧颈动脉内可探测到的血管斑块数量密
切相关,说明胱抑素 C 水平越高颈动脉斑块越多;多元相关分
析及 Logistic 回归显示,胱抑素 C 在早期 2 型糖尿病颈动脉内
膜增厚中的影响不容忽视,其危险度较 2hPG、HBA1C、hs-CRP 更
强。
刘姝等[13]研究显示,2 型糖尿病各组间血清胱抑素 C 水平
随下肢动脉病变程度的加重而逐渐升高,2 型糖尿病伴下肢动
脉中、重度闭塞组显著高于正常对照组,伴下肢动脉病变组胱
抑素 C 水平显著高于不伴下肢动脉病变组;Logistic 回归分析
发现胱抑素 C 与 2 型糖尿病下肢动脉病变显著相关,为 2 型糖
尿病下肢动脉病变的高危因素,糖尿病患者,胱抑素 C 升高提
示下肢动脉病变更严重。
在糖尿病人群中,尚无关于胱抑素 C 与脑血管疾病关系的
研究。
四、 胱抑素 C 与糖尿病微血管病变
由于胱抑素 C 相对分子质量低,等电点高(9. 3),能被肾小
球自由滤过,然后在近曲小管上皮细胞重吸收并完全分解代
谢,不返回血液,而且肾小管上皮细胞也不分泌胱抑素 C 至管
腔内,血液中的胱抑素 C 浓度主要由 GFR 决定,所以胱抑素 C
被认为是评估 GFR 较理想的内源性标志物,其作用甚至优于
血肌酐(Scr) [14,15] 。 Tsai 等[7]分析了 1999 至 2002 年美国国家
卫生和营养调查中的 778 例糖尿病患者,以 Scr 估算肾小球滤
过率(eGFRScr)和 eGFR胱抑素C评估肾脏功能得到的肾功能不全
的患病率分别为 16. 5%和 22. 0% ,提示 eGFR胱抑素C对肾功能的
评估更敏感。 Suzuki 等[16]对 2 型糖尿病研究发现,在糖尿病肾
病第 2 期[尿微量白蛋白 /肌酐(MA / Cre)30 ~ 300 mg / g]患者
中胱抑素 C 明显增高,而 Scr、血清 β2-微球蛋白(sβ2 m)却无明
显变化,提示血清胱抑素 C 在早期诊断糖尿病肾病的敏感性及
特异性均明显优于 Scr 和 sβ2 m,并由此推断血清胱抑素 C 可
以有效检测出早期糖尿病肾病,并对慢性肾脏病 2 期的筛选可
能非常有用。 另外,Chae 等[17]在儿童和青少年(12 ~ 19 岁)糖
尿病人群中发现相对于 24 h 尿白蛋白排泄率和 Scr,联合检测
随机尿 MA / Cre 和血清胱抑素 C 能更好地预测儿童和青少年
糖尿病肾病(DN)的发生以及肾损害程度。 Pucci 等[18] 在 288
例糖尿病人群中(T1DM 125 例,2 型糖尿病 163 例)发现胱抑素
C 与碘海醇清除率(iGFR)的相关性最好,尤其在 T1DM 患者和
肾功能正常者,与 Scr 及 eGFRScr相比,检测胱抑素 C 能更好地
评价 1 型和 2 型糖尿病早期肾损害( iGFR 在 70 ~ 90 ml / min·
1. 73 m2之间)。 与之相反的是,Iliadis 等[19]以51CrEDTA 清除率
为标准评价肾小球滤过率(GFR),在 2 型糖尿病人群中比较
eGFR胱抑素C和 eGFRSCr的优劣,发现 eGFR胱抑素C对预测 CKD 分期
的效果并没有 eGFRSCr好。 但 Bevc 等[20] 同样以51CrEDTA 清除
率为标准评价 GFR,在超重(BMI>25 kg / m2 )且合并慢性肾脏
疾病的 2 型糖尿病患者中发现:eGFR胱抑素C公式可能是评估超
重且合并 CKD 的 2 型糖尿病患者肾功能以及住院、尤其门诊
患者肾损伤的有力工具,虽然它并不能完全替代评估 GFR 的
金标准,但有助于临床医生准确筛选出需要侵入性或昂贵操作
的患者。
陈兰英等[21]研究显示,2 型糖尿病伴视网膜病变患者的血
清胱抑素 C 水平明显高于 2 型糖尿病不伴视网膜病变者;
Logistic 多元逐步回归分析表明血清胱抑素 C 是影响 2 型糖尿
病患者视网膜病变发生的独立危险因素,提示检测血清胱抑素
C 水平对糖尿病视网膜病变的早期诊断和防治可能具有一定
的临床指导意义。 He 等[22] 的研究进一步显示胱抑素 C 的升
高与糖尿病视网膜病变的严重程度正相关,血清胱抑素 C 每升
高 1. 25 mg / L,重度视网膜病变的风险升高 11 倍,因此,胱抑素
C 是预测糖尿病严重视网膜病变的高效廉价的血清学指标。
高胱抑素 C 可引起炎症反应,同时又能抑制半胱氨酸蛋白酶活
性,升高 C 反应蛋白(CRP)、同型半胱氨酸(Hcy)浓度,但胱抑
素 C 是否存在对微血管的特异性损害,是直接或通过 CRP 和
Hcy 间接对微血管内皮产生影响,促进 DR 的发生与发展目前
尚不清楚。
有研究发现,糖尿病心肌病患者的血清胱抑素 C、肌钙蛋
白 I 明显高于健康者,提示血清胱抑素 C 可能对糖尿病心肌病
的临床诊治有一定的指导意义[23] 。 研究发现,视网膜色素上
皮细胞也是胱抑素 C 分泌的重要细胞,并证实胱抑素 C 在年龄
相关性黄斑变性中有着重要作用[24] 。 目前尚无关于血清胱抑
素 C 和糖尿病周围神经病变的报道,结合前述血清胱抑素 C 与
糖尿病肾病、糖尿病视网膜病变的关系,推测在糖尿病周围神
经病变患者中血清胱抑素 C 可能亦呈高水平状态,但需要进一
步临床研究加以证实。
五、 结语
胱抑素 C 参与糖尿病及糖尿病大血管、微血管并发症的诸
多病理生理过程,但机制尚不清楚,可能涉及抑制半胱氨酸蛋
白酶的活性、胰岛素抵抗、内皮细胞损伤、炎症等,系未来的研
究方向之一。
