临床研究
2 型糖尿病患者血清胆红素与血清 C 肽水平的关系
杨成会  徐家蓉  游娜  缪珩
[提要]   探讨 2 型糖尿病患者血清胆红素与血清 C 肽水平之间的关系。 收集并分析 206 名健康对照
者和 271 例 2 型糖尿病患者一般临床指标及生化指标,对血清胆红素与血清 C 肽水平进行偏相关与多元
线性回归分析,发现血清胆红素水平与空腹 C 肽、餐后 C 肽、餐后 C 肽和空腹 C 肽的差值呈正相关(β =
0. 171、0. 111、0. 052,P<0. 05 或 P<0. 01)。
[关键词]   糖尿病, 2 型; 胆红素; β 细胞; C 肽; 氧化应激
Relationship between serum bilirubin and serum C-peptide levels in patients with type 2 diabetes mellitus
Yang Chenghui, Xu Jiarong, You Na,Miao Heng. The Department of Endocrine and Metabolism, the Second Hospital
of Nanjing Medical University, Nanjing 210000, China
[Summary]  The relationship between serum bilirubin and serum C-peptide levels in patients with type 2
diabetes mellitus was investigated. The clinical and laboratory data of 206 healthy subjects and 271 type 2 diabetic
patients were collected. The partial correlation analysis and multiple linear regression analysis showed that serum
bilirubin was positively correlated with fasting C-peptide, postprandial C-peptide, and the difference between
postprandial C-peptide and fasting C-peptide levels (β=0. 171, 0. 111,0. 052,P<0. 05 or P<0. 01) .
[Key words]  Diabetes mellitus, type 2; Bilirubin; β-cell; C-peptide; Oxidative stress
(Chin J Endocrinol Metab, 2016, 32: 298-299)
近年来,胆红素作为内源性抗氧化剂通过清除氧自由基对
抗氧化应激反应逐渐被临床所认识[1] 。 目前关于血清胆红素
与胰岛 β 细胞的临床研究尚少,本研究旨在观察血清胆红素与
血清 C 肽水平之间的关系,并探讨其发挥的作用。
一、 对象和方法
1. 对象:选取 2014 年 1 月至 2014 年 9 月于南京医科大学
第二附属医院内分泌科住院的 2 型糖尿病患者 271 例,其中男
性 142 例,女性 129 例;正常对照组 206 名。 所有受试者均排除
心、脑、肝、胰腺、肾、血液等严重疾病,近期未服用任何影响胆
红素代谢的药物、无急性感染等应激情况。
2. 方法:所有患者均食用按体重计算的标准餐,并于餐后
2h 抽取静脉血测餐后血清 C 肽水平,采用放射免疫法检测;
HBA1C检测采用高效液相色谱法;空腹血糖(FPG)、血清三酰甘
油(TG)、总胆固醇 ( TC)、谷丙转氨酶 ( ALT)、谷草转氨酶
(AST)、低密度脂蛋白胆固醇(LDL-C)、高密度脂蛋白胆固醇
(HDL-C)、总胆红素(TBIL)、直接胆红素(DBIL)、间接胆红素
(IBIL)均采用迈瑞全自动生化分析仪检测;体重指数(BMI)=
体重 /身高2(kg / m2)。
3. 统计学处理:应用 SPSS 19. 0 分析软件,2 组间比较采用
t 检验,多组间比较采用方差分析;血清胆红素与 C 肽之间的关
系采用偏相关分析和多元线性回归分析。 P<0. 05 为差异有统
计学意义。
二、 结果
1. 正常对照组与 2 型糖尿病组临床资料比较:2 型糖尿病
组患者的 BMI、高血压患病率、高血脂患病率、收缩压、舒张压、
FPG、HBA1C、TG 均显著高于正常对照组, TBIL、 DBIL、 IBIL、
HDL-C 显著低于对照组(均 P<0. 05)。
2. 不同血清胆红素水平 2 型糖尿病患者的临床特征比较:
2 型糖尿病患者根据 TBIL 水平分为组 1(TBIL≤9. 9 μmol / L)
和组 2(TBIL>9. 9 μmol / L),组 2 较组 1 糖尿病病程短、HBA1C
水平低,而 BMI、空腹 C 肽、餐后 C 肽、吟C 肽(餐后 C 肽与空腹
C 肽的差值)水平较高(P<0. 05 或 P<0. 01,表 1)。
3. 血清 C 肽水平与临床变量间的偏相关分析:将空腹 C
肽、餐后 C 肽、吟C 肽变量分别与 TBIL、BMI、HBA1C、糖尿病病
程进行偏相关分析,结果显示 TBIL、BMI 与空腹 C 肽、餐后 C
肽、吟C 肽水平间呈正相关,HBA1C、糖尿病病程与空腹 C 肽、餐
后 C 肽、吟C 肽水平间呈负相关(P<0. 05 或 P<0. 01,表 2)。
4. 2 型糖尿病患者以血清 C 肽水平作为因变量的多元线
性回归分析:分别以空腹 C 肽、餐后 C 肽、△C 肽为因变量,以
TBIL、BMI、HBA1C、糖尿病病程作为自变量进行多元线性回归
分析,发现 TBIL 与空腹 C 肽、餐后 C 肽、△C 肽水平均呈正相
关,标准偏回归系数分别为 0. 171、0. 111、0. 052;HBA1C、糖尿病
病程与其呈负相关(P<0. 05 或 P<0. 01)。
三、 讨论

生发展中起重要生理意义。 在一项大型前瞻性研究中发
现,TBIL>10 μmol / L 的人群发生 2 型糖尿病的风险比 TBIL<10
μmol / L 的人群低 20% [2] 。 在本项研究中,2 型糖尿病组与正

常对照组相比较,TBIL、DBIL、IBIL 水平均显著低于健康人群,
提示高胆红素可能是发生 2 型糖尿病的保护因素。 另有研究
显示适量的胆红素浓度对糖尿病大血管病变及微血管病变起
到保护作用[3-5] 。
多项研究表明,氧化应激是导致胰岛 β 细胞功能衰退的重
要因素[6] ,活性氧簇(ROS)主要通过直接损伤 β 细胞线粒体结
构和激活 NFκB 信号通路,引起 β 细胞炎症反应和凋亡[7] ,还
可抑制胰腺十二指肠同源盒-1(PDX-1)的核转位和肌腱膜纤维
肉瘤肿瘤基因同系物 A(MafA)的转录、翻译过程,减少胰岛素
的合成与分泌等[8] 。 在本研究中 TBIL 水平较高的患者,其糖
尿病病程、HBA1C水平较低,而 BMI、血清 C 肽水平较高。 在偏
相关与多元线性回归分析中,TBIL、BMI 与血清 C 肽呈正相关,
HBA1C、糖尿病病程与其呈负相关,说明血清胆红素可能通过抗
氧化对胰岛 β 细胞起到保护作用。 另外 Kapitulnik 等[9]发现在
高糖导致氧化应激的条件下,胆红素通过清除 ROS 对胰岛微
血管内皮细胞产生保护作用。 Ikeda 等[10] 用胆红素的前体物
质胆绿素喂养 db / db 小鼠 4 周,发现胆绿素可以防止高血糖和
糖耐量受损的恶化,并且 PDX-1 的表达和胰岛素含量增加,胰
岛 β 细胞凋亡和 Bax 表达减少,同时氧化应激标记物 8-羟基脱
氧鸟苷和 NADPH 氧化酶活性均降低。 因此,我们推测胆红素
可能通过参与氧化应激反应减轻胰岛 β 细胞的损伤。
综上所述,本研究发现血清胆红素与血清 C 肽水平呈正相
关,提示我们在临床工作中应注意血清胆红素水平与氧化应激
的关系,积极抗氧化治疗,可能有助于避免胰岛 β 细胞的进一
步损伤,有助于延缓糖尿病的进展。 本研究仅能表明血清胆红
素与 C 肽是伴随关系,尚不能推断有因果关系,仍需要进一步
的临床研究。
